# Node-Exporter-Automater
This Script will automaticaly install node-exporter (bare-metal) on a machine 
Useful if you are installing it on multiple machines

Node Expoter Github:
https://github.com/prometheus/node_exporter

# Usage
Download and Run:
curl -fsSL https://raw.githubusercontent.com/johanjw4/Node-Exporter-Automater/main/nodeexporterauto.sh | bash

Only Download
curl -fsSL https://raw.githubusercontent.com/johanjw4/Node-Exporter-Automater/main/nodeexporterauto.sh -o nodeexporterauto.sh
